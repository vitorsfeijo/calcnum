# README

### How to create venv

1. python3 -m venv .venv
2. source .venv/bin/activate

### How to install requirements

pip install -r requirements.txt

### How to take out the warnings from global python

1. crtl + shift + p
2. python: select interpreter
3. select the venv one
4. crtl + shift + p
5. reload window
